Text to Colour Converter

This tool converts text into a Hexadecimal value, which is then displayed as a series of HTML colours. This can be a way to package information in a different way (See Illegal numbers)

  • (Note. As HTML colours consist of 6 digit hexadecimals, incomplete hexadecimals are rounded with ‘0’s to the nearest colour.)
